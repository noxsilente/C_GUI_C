## **English**
Thank you for your interest in **C_GUI**, an open‑source project developed in **C# using Visual Studio**.  
This application provides an informational catalog of ammunition, organized into categories such as **pistol/revolver**, **rifle**, and **large calibers**. All data has been collected from publicly available sources and is intended solely for reference and educational purposes.

Images included in the project are either **public domain** or created specifically for the application and marked with the project logo.

This repository contains the **source code** of the project.  
The working executable and full data files are available in the *Release* section.
