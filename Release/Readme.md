## **English**
Thank you for downloading the **C_GUI** ready‑to‑use release.  
This version includes the executable, configuration files, ammunition parameters, and images required for full functionality.

C_GUI is an open‑source informational catalog of ammunition, organized into **pistol/revolver**, **rifle**, and **large‑caliber** categories. All data is collected from publicly available sources and provided for reference only.

Images included in this release are **public domain** or created for the application and marked with the project logo.

If you encounter issues or wish to contribute, please visit the main repository.
